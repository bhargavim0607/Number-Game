HOW TO PLAY

This game consists of 25 tiles.
We need to fill all 25 Tiles from 1 to 25 numbers by following below rules:

  1. Place the First Number on any one of the tile.
  2. After the first number some of the tiles gets blue color indicating that you can place next number only in those tiles.
  3. You will not be able to place the number in tile which is not indicated.
  4. The tiles are indicating in blue color according to following:
    • If you wanted to move in vertical or horizantal then you need to leave 2 tiles in middle of start and end tile
    • Similarly to move diagonal leave one tile
  5. By following all these rules if you can fill all tiles with all 25 numbers then you will be the winner.üèÜ
